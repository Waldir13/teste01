    Fontes externas

    quando for baixar a fonte observar se tem ministula e acentos.
    Entra no dafonte baixa, a pasta vem zipada, clica com o direito e coloca extrair tudo, entra na pasta e copia os arq para a pasta que esta usando, vai para o css e add @fonte-face, na url coloca o nome do arq,depois add format e informa o formato.
    Se a fonte vir com dois formatos é bom add os dois, caso algum navegador não reconheça alguns dos formatos, é só add outra url. obs: para ver o formato da fonte, é so clicar c/ o direito, em propriedades.
    Não esquecer de colocar um nome na font-family, e depois colocar esse nome na tag que eu quero usar
